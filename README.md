# Industry 4.0 - Digital Production Monitoring System

## Project Goal

The goal of this project is to design and implement a simple digitalization solution for monitoring a manufacturing production line using Industry 4.0 principles.

## Problem Description

Manufacturing companies need real-time visibility into producciton performance, machine downtime, and efficiency indicators in order to improve productivity and reduce losses.

## Solution Overview

This project simulates production line data and provides:

- Collection of machine data
- Storage in a relational database
- Calculation of basic KPIs (e.g. OEE)
- Visualization in a simple dashboard

## Target Users

- Production managers
- Process engineers
- Digitalization teams

## Technology Stack (planned)

- Backend: Python or C#
- Databasse: SQL (PostrgeSQL or MSSQL)
- Frontend: Simple web dashboard
- Visualization: Basic charts
